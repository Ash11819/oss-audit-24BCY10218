## Project Overview

This project is an audit of **Mozilla Firefox**, a widely used open-source web browser developed by the **Mozilla Foundation**.

The aim of this project is to understand:

* The origin and philosophy behind Firefox
* Its open-source license and ethical values
* Its role in the Linux ecosystem
* Comparison with proprietary browsers

This project also includes **five shell scripts** that demonstrate practical Linux and open-source concepts.

---

##  About Mozilla Firefox

Mozilla Firefox is a free and open-source web browser that focuses on:

* User privacy
* Transparency
* Community-driven development

It is one of the few browsers not controlled by a large corporation, making it an important part of the open web ecosystem.

##  Shell Scripts Description

### 🔹 Script 1 – System Identity Report

Displays:

* Kernel version
* Logged-in user
* System uptime
* Date and time

Demonstrates variables and command substitution.

---

### 🔹 Script 2 – FOSS Package Inspector

* Checks if Firefox is installed
* Displays version and details

 Uses if-else, case statements, and package management commands.

---

### 🔹 Script 3 – Disk and Permission Auditor

* Analyzes important system directories
* Shows size, ownership, and permissions

 Uses loops and Linux commands.

---

### 🔹 Script 4 – Log File Analyzer

* Reads log files
* Counts occurrences of keywords like "error"

 Demonstrates loops and conditional logic.

### 🔹 Script 5 – Open Source Manifesto Generator
